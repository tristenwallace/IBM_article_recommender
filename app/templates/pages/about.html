About the Article Recommender Project

This project aims to recommend relevant articles to users based on their reading history and preferences, leveraging advanced data processing and machine learning techniques. The system also enriches article recommendations with visually appealing images sourced from Unsplash.

Tech Stack

Flask

The web framework used for building the application, providing the necessary infrastructure for web requests, routing, and templating.

Pandas

Utilized for efficient data manipulation and analysis, particularly for handling the articles and user interactions datasets.

Scikit-learn

Empowers the recommender system with machine learning capabilities, including TF-IDF vectorization and cosine similarity for article recommendations.

NLTK

Used in the image search functionality to process and refine article titles into keywords for improved image search relevance.

Unsplash API

Provides access to high-quality images to enhance the article recommendations with visually relevant imagery.

How It Works

The Article Recommender system analyzes user-article interaction data to understand user preferences. It then creates a profile for each article using TF-IDF vectorization, capturing the essence of articles based on their titles and descriptions.

Recommending Articles

For a given user, the system computes cosine similarity between articles the user has interacted with and all other articles to identify and recommend the most similar, unread articles.

Enhancing Recommendations with Images

Using refined keywords extracted from article titles, the system queries the Unsplash API to fetch relevant images, adding a visual dimension to the recommendations presented to the user.
